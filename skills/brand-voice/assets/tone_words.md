# Tone-of-voice words

The 37-word tone vocabulary from Nielsen Norman Group ("Tone-of-Voice Words"),
the standard purpose-built set for a brand-voice card sort. Use this exact list
so the elicitation is identical every run — present it verbatim, don't improvise
or substitute synonyms.

Authoritative, Caring, Casual, Cheerful, Coarse, Conservative, Conversational,
Dry, Edgy, Enthusiastic, Formal, Frank, Friendly, Fun, Funny, Humorous,
Informative, Irreverent, Matter-of-fact, Nostalgic, Passionate, Playful,
Professional, Provocative, Quirky, Respectful, Romantic, Sarcastic, Serious,
Smart, Snarky, Sympathetic, Trendy, Trustworthy, Unapologetic, Upbeat, Witty.
